Customer Complaint Root Cause Analysis using NLP

Objective

The goal of this project is to analyze customer complaint narratives and identify the root causes behind complaints across different financial products using Natural Language Processing (NLP) techniques.

Dataset Source: CFPB Consumer Complaint Database (public dataset).
The raw CSV file is not included in this repository due to its large size.

Conclusion

This project analyzed real-world customer complaint narratives to identify the root causes behind complaints across financial products. By applying TF-IDF vectorization and a Naive Bayes model, key complaint drivers such as billing issues, overdraft fees, late payments, inaccurate credit reporting, and debt collection practices were identified. These insights can help organizations prioritize process improvements and reduce future complaints.
